08CMS 软件安装说明

 

感谢您选择 08CMS 网站内容管理系统（以下简称08CMS），以下为08CMS安装简要说明，更多内容请参照官方网站www.08cms.com中的相关详细说明。

1、安装环境要求

08CMS 是基于 PHP + MYSQL 的跨平台的系统，可以在 Linux/FreeBSD/Unix 及微软 Windows 2003/2008 等各种操作系统运行。

需要安装 Apache 或 IIS 或 NGINX 等 WEB 服务器、php 5.2.3 或以上版本、Zend Optimizer 3.0 或以上版本、MySQL 5.0.7 或以上版本的软件。

2、安装文件上传

软件包中的 upload 目录及所含目录及文件，为安装时必须上传至空间的内容。

通过 FTP 软件上传安装文件时，请使用二进制（BINARY）模式进行，否则将无法正常使用。

3、目录属性设置

请确认以下目录或文件权限为可读写（以*结尾的目录必须将其子目录及文件一并设置）。

       系统根目录 ./

       系统配置文件 ./base.inc.php

       缓存目录 ./dynamic/*

       静态文件目录 ./html/*

       会员频道目录 ./userfiles/*

       模板目录 ./template/*

       附件上传目录 ./userfiles/*

如果您的FTP有权限的话，您可以通过 FTP 工具或远程登录主机对目录或文件的属性进行修改。
