flash列表 file
概述

flash列表用来调用模型字段类型为[Flash集]的数据。

参数

1. 指定内容来源只允许为Flash集类型字段。

2. 本标识属列表性质标识，列表展示Flash集的多个Flash内容。

3. {url}调用Flash的URL，{title}调用Flash标题,{playbox}调用该Flash的播放模块。

4. 专用播放页面链接{$cms_abs}tools/flash.php?tname=xxxx&aid={aid}&fid={fid}，其中xxxx为Flash集的字段名称，{fid}为集内的序号。
