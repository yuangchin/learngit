空间类目导航 mnownav
概述

空间类目导航是系统标签之一，不常用。

  • 效果:
  • 标识模板(template)代码:
  • <a href="{indexurl}">{title}</a>
  • 说明：

  • 1. 本标识限用于个人空间的类目页面，显示当前页的类目位置导航。
    2. 类目属性来源于当前页面的类目属性。
    3. 标识模板允许内嵌通用原始信息及类目相关原始标识及相关的其它原始标识。
    4. 标识内置参数：{indexurl} 标识表示节点链接地址，标识内的indexurl后可加一个正整数表示第X个节点附加页节点地址。
