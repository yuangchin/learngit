flash模块 file
概述

flash模块是调用模型字段类型为[Flash]的数据。

用法说明
1. 内容来源应为Flash类型。当本标识直接用于flash.php的模板中时，内容来源留空。
2. {url}用于调用附件url，{playbox}用于调用该Flash的播放模块。
3. 播放器宽度或播放器高度为0时，将会以系统在〖系统设置〗→〖网站参数〗→〖附件设置〗里的媒体设置的大小显示。
